# Kencleng Digital - Aplikasi Pengelolaan Infak Berbasis Desa

Aplikasi digital untuk mengelola program kencleng (celengan) infak di tingkat desa, terinspirasi dari kesuksesan BAZNAS Ciamis yang berhasil mengumpulkan Rp22 miliar dalam setahun.
